== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/responsive-post-slider` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Responsive Post Slider screen to configure the plugin
